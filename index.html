Understanding the CSS Box Model

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

Margin

Margin is the space outside the border. It creates space between elements.

Border

Border goes around the padding and content. You can style it in many ways.

Padding

Padding is the space between the content and the border.

Applying CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. With CSS, you can control the color, font, size of text, spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!
